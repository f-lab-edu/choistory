# choistory
시간이 지나면 선택한 기록 외에는 모두 지워지는 sns

## Feature
### 서비스시작
- 회원가입
- 로그인
### 구독(Follow)
- Follow
- Unfollow
### 타임라인
#### 글 쓰기
- 이미지 올리기
- 남길 글 선택
- 유효시간 선택
#### 글 삭제
#### 글 읽기
- 구독 글까지 시간순 정렬
- 답글달기
